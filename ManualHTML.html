🖥️ Conceptos Básicos

HTML (HyperText Markup Language) es el estándar para la creación de páginas web. Permite estructurar texto, imágenes y enlaces.

🌟 Etiquetas Fundamentales:

  • <html>: Marca el inicio del documento.
  • <head>: Contiene metadatos y enlaces externos.
  • <body>: Incluye todo el contenido visible.

🎯 Ejemplo Práctico:

<html>
  <head>
    <title>Mi Página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Esto es HTML básico.</p>
  </body>
</html>
